## ¿Qué son los formularios?

Los **formularios** son interfaces que permiten a los usuarios ingresar, editar y enviar información al sistema. En PathSys, los formularios están diseñados para ser intuitivos, accesibles y proporcionar una experiencia consistente en toda la aplicación.

### ¿Por qué son importantes?

- **Captura de datos**: Permiten ingresar información de pacientes, casos y pruebas
- **Validación en tiempo real**: Proporcionan feedback inmediato sobre errores
- **Experiencia de usuario**: Guían al usuario en el proceso de llenado
- **Consistencia**: Mantienen un diseño uniforme en toda la aplicación

## Ubicación de los formularios en el código

Los componentes de formularios en PathSys están organizados en una estructura modular:

### Estructura de archivos

```
Front-End/src/shared/components/ui/forms/
├── FormInput.vue              ← Campo de texto básico
├── FormInputField.vue         ← Campo de texto con validación
├── FormSelect.vue             ← Selector desplegable
├── FormCheckbox.vue           ← Casillas de verificación
├── FormTextarea.vue           ← Área de texto
├── FormTextareaUnlimited.vue  ← Área de texto sin límite
├── DateInputField.vue         ← Campo de fecha
├── FormButton.vue             ← Botón de formulario
├── RichTextEditor.vue         ← Editor de texto enriquecido
└── index.ts                   ← Exportaciones centralizadas
```

Los campos de texto son para ingresar información básica:

- **Etiqueta clara**: Identifica qué información se debe ingresar
- **Placeholder**: Proporciona ejemplo o instrucción
- **Validación visual**: Borde rojo para errores, verde para éxito
- **Iconos opcionales**: Para clarificar el tipo de dato

**¿Cuándo usar campos de texto?**
- Nombres y apellidos
- Números de identificación
- Direcciones y datos de contacto
- Cualquier información alfanumérica

**Ejemplo de uso:**
<CodeBlock language="vue">{`<template>
  <FormInput
    v-model="patientName"
    label="Nombre del paciente"
    placeholder="Ingrese el nombre completo"
    :required="true"
    :error="nameError"
    help-text="Ingrese nombre y apellidos"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { FormInput } from '@/shared/components/ui/forms'

const patientName = ref('')
const nameError = ref('')
</script>`}</CodeBlock>

<img 
  src="/img/Captures/CampoSelector.png"

  <strong>Campo selector:</strong> Lista desplegable para seleccionar entre opciones predefinidas.

Los campos selector permiten elegir entre opciones específicas:

- **Opciones predefinidas**: Lista de valores válidos
- **Búsqueda**: Filtrado de opciones para listas largas
- **Selección múltiple**: Opción de elegir varios valores
- **Validación**: Asegura que se seleccione una opción válida

**¿Cuándo usar campos selector?**
- Selección de entidades o instituciones
- Categorías y clasificaciones
- Estados y tipos predefinidos
- Cualquier lista de opciones limitadas

**Ejemplo de uso:**
<CodeBlock language="vue">{`<template>
  <FormSelect
    v-model="selectedEntity"
    label="Entidad"
    placeholder="Seleccione una entidad"
    :options="entityOptions"
    :required="true"
    :error="entityError"
    help-text="Seleccione la entidad de salud"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { FormSelect } from '@/shared/components/ui/forms'

const selectedEntity = ref('')
const entityError = ref('')
const entityOptions = [
  { value: 'eps-001', label: 'EPS Sanitas' },
  { value: 'eps-002', label: 'EPS Sura' },
  { value: 'eps-003', label: 'EPS Coomeva' }
]
</script>`}</CodeBlock>

<img 
  src="/img/Captures/Checkbox.png"

  <strong>Casillas de verificación:</strong> Para seleccionar múltiples opciones o confirmar condiciones.

Las casillas de verificación permiten selecciones múltiples:

- **Selección múltiple**: Elegir varias opciones simultáneamente
- **Estados binarios**: Sí/No, Activo/Inactivo
- **Agrupación**: Múltiples opciones relacionadas
- **Validación**: Al menos una opción seleccionada

**¿Cuándo usar casillas de verificación?**
- Síntomas o condiciones médicas
- Servicios adicionales
- Permisos y configuraciones
- Cualquier selección múltiple

**Ejemplo de uso:**
