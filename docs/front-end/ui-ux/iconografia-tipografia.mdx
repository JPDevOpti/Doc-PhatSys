## ¿Qué es la iconografía y tipografía?

La **iconografía** y **tipografía** son elementos fundamentales del diseño de PathSys que establecen la identidad visual y mejoran la usabilidad del sistema. La iconografía proporciona símbolos universales para acciones y conceptos, mientras que la tipografía asegura la legibilidad y jerarquía de la información.

### ¿Por qué son importantes?

- **Consistencia visual**: Mantienen un lenguaje visual uniforme en todo el sistema
- **Usabilidad**: Los iconos facilitan la navegación y comprensión rápida
- **Legibilidad**: La tipografía optimizada mejora la experiencia de lectura
- **Identidad de marca**: Refuerzan la imagen profesional del sistema LIME

### Características del sistema de iconos

**Estilo consistente:**
- **Líneas delgadas**: Grosor uniforme de 1.5px para todos los iconos
- **Sin rellenos**: Solo contornos para mantener simplicidad
- **Color uniforme**: Uso de `currentColor` para adaptabilidad
- **Tamaño estándar**: ViewBox de 24x24px para todos los iconos
- **Diseño minimalista**: Sin elementos decorativos innecesarios

**Categorías de iconos:**
- **Navegación**: Flechas, menús, home, dashboard
- **Acciones**: Editar, eliminar, agregar, guardar, imprimir
- **Usuarios**: Perfil, grupo de usuarios, búsqueda de usuarios
- **Documentos**: Archivos, listas, tablas, gráficos
- **Estados**: Éxito, error, advertencia, información
- **Médicos**: Casos, pacientes, pruebas, entidades hospitalarias

## Tipografía en PathSys

### Fuente principal: Outfit

PathSys utiliza la fuente **Outfit** de Google Fonts como tipografía principal:

**Características de Outfit:**
- **Familia**: Sans-serif moderna y legible
- **Pesos disponibles**: 100-900 (todas las variantes)
- **Estilo**: Diseño geométrico y limpio
- **Optimización**: Cargada desde Google Fonts con `display=swap`

### Configuración tipográfica

<CodeBlock language="css">{`/* Importación desde Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Outfit:wght@100..900&display=swap');

/* Variables CSS para tipografía */
:root {
  --font-outfit: Outfit, sans-serif;
  
  /* Tamaños de texto y line-height */
  --text-title-2xl: 72px;
  --text-title-2xl--line-height: 90px;
  --text-title-xl: 60px;
  --text-title-xl--line-height: 72px;
  --text-title-lg: 48px;
  --text-title-lg--line-height: 60px;
  --text-title-md: 36px;
  --text-title-md--line-height: 44px;
  --text-title-sm: 30px;
  --text-title-sm--line-height: 38px;
  --text-theme-xl: 20px;
  --text-theme-xl--line-height: 30px;
  --text-theme-sm: 14px;
  --text-theme-sm--line-height: 20px;
  --text-theme-xs: 12px;
  --text-theme-xs--line-height: 18px;
}

/* Aplicación global */
body {
  font-family: var(--font-outfit);
  font-weight: 400;
  background-color: #f9fafb;
}`}</CodeBlock>

### Jerarquía tipográfica

**Títulos principales (Títulos de página):**
- **Tamaño**: 36px (--text-title-md)
- **Line-height**: 44px
- **Peso**: 600-700 (semi-bold/bold)
- **Uso**: Nombres de módulos, páginas principales

**Títulos secundarios (Subtítulos):**
- **Tamaño**: 30px (--text-title-sm)
- **Line-height**: 38px
- **Peso**: 500-600 (medium/semi-bold)
- **Uso**: Secciones dentro de módulos

**Texto temático (Contenido destacado):**
- **Tamaño**: 20px (--text-theme-xl)
- **Line-height**: 30px
- **Peso**: 400-500 (regular/medium)
- **Uso**: Textos importantes, etiquetas principales

**Texto estándar (Contenido general):**
- **Tamaño**: 14px (--text-theme-sm)
- **Line-height**: 20px
- **Peso**: 400 (regular)
- **Uso**: Párrafos, descripciones, contenido principal

**Texto pequeño (Información secundaria):**
- **Tamaño**: 12px (--text-theme-xs)
- **Line-height**: 18px
- **Peso**: 400 (regular)
- **Uso**: Fechas, códigos, información auxiliar

## Mejores prácticas

### Iconografía

**Consistencia:**
- Usar siempre iconos del sistema unificado
- Mantener el mismo tamaño dentro de un contexto
- Aplicar colores consistentes con el tema

**Accesibilidad:**
- Incluir `aria-hidden="true"` en iconos decorativos
- Proporcionar texto alternativo cuando sea necesario
- Mantener contraste adecuado

**Usabilidad:**
- Usar iconos universalmente reconocidos
- Combinar iconos con texto cuando sea apropiado
- Mantener tamaños mínimos de 16px para interacción

### Tipografía

**Legibilidad:**
- Mantener line-height de al menos 1.4
- Usar contraste mínimo de 4.5:1
- Limitar líneas de texto a 75-80 caracteres

**Jerarquía:**
- Establecer jerarquía clara con tamaños
- Usar pesos de fuente para enfatizar
- Mantener consistencia en espaciado

**Responsividad:**
- Ajustar tamaños para diferentes pantallas
- Mantener proporciones en dispositivos móviles
- Optimizar para lectura en pantalla
