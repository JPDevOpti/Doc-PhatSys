## ¿Qué es UI & UX en PathSys?

**UI & UX** (User Interface & User Experience) en PathSys abarca todo el sistema de diseño, componentes de interfaz y patrones de experiencia de usuario que conforman la aplicación. Incluye desde la estructura visual básica hasta los detalles de interacción, asegurando una experiencia consistente, accesible y eficiente para todos los usuarios del sistema.

### ¿Por qué es importante?

- **Consistencia visual**: Mantiene un lenguaje de diseño uniforme en toda la aplicación
- **Eficiencia de usuario**: Optimiza los flujos de trabajo para profesionales médicos
- **Accesibilidad**: Asegura que el sistema sea usable por todos los tipos de usuarios
- **Mantenibilidad**: Componentes reutilizables que facilitan el desarrollo y actualizaciones

## Sistema de diseño LIME

PathSys implementa un sistema de diseño coherente basado en la identidad institucional de LIME (Laboratorios Integrados de Medicina Especializada), con elementos visuales que reflejan la profesionalidad y confiabilidad requerida en el ámbito médico.

### Principios fundamentales

1. **Claridad**: Interfaces limpias que facilitan la toma de decisiones médicas
2. **Eficiencia**: Flujos optimizados para reducir el tiempo de trabajo
3. **Confianza**: Diseño profesional que inspira confianza en los usuarios
4. **Accesibilidad**: Cumplimiento de estándares de accesibilidad web

## Cómo usar el sistema de diseño

### Para desarrolladores

El sistema de diseño de PathSys está implementado con **Vue 3** y **Tailwind CSS**, proporcionando componentes reutilizables y utilidades de estilo consistentes.

**Ubicación de componentes:**
```
src/shared/components/ui/
├── buttons/          ← Componentes de botones
├── forms/            ← Componentes de formularios
├── lists/            ← Componentes de listados
├── feedback/         ← Componentes de feedback
└── navigation/       ← Componentes de navegación
```

**Importación de componentes:**
<CodeBlock language="vue">{`<script setup lang="ts">
// Importar componentes individuales
import { BaseButton, FormInput, Notification } from '@/shared/components/ui'

// O importar desde el índice
import { BaseButton, FormInput } from '@/shared'
</script>

<template>
  <BaseButton variant="primary" size="md">
    Guardar
  </BaseButton>
  
  <FormInput 
    v-model="value" 
    label="Nombre"
    placeholder="Ingrese el nombre"
  />
</template>`}</CodeBlock>

Este sistema de diseño asegura que PathSys mantenga una experiencia de usuario profesional, consistente y eficiente, optimizada para el trabajo médico especializado.
