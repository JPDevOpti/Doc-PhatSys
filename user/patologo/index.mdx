# Bienvenido al Módulo de Patología

PathSys es el sistema de gestión de información de laboratorio diseñado para optimizar el flujo de trabajo en el laboratorio de patología de la UT LIME.

Como patólogo, este sistema le permitirá:
- Gestionar sus casos asignados de manera eficiente.
- Registrar hallazgos macroscópicos y microscópicos.
- Solicitar pruebas complementarias y visualizar resultados.
- Generar y firmar reportes diagnósticos estandarizados.

### Pasos para iniciar sesión

1. Ingrese a la URL del sistema proporcionada por el administrador.
2. En la pantalla de inicio de sesión, introduzca su **nombre de usuario** y **contraseña**.
3. Haga clic en el botón **"Ingresar"**.
4. Al acceder, será redirigido a su panel principal (Dashboard) donde verá sus casos pendientes y métricas relevantes.

:::tip Nota de Seguridad
Si es su primer ingreso, el sistema le solicitará cambiar su contraseña temporal por una definitiva. Asegúrese de no compartir sus credenciales con nadie.
:::
