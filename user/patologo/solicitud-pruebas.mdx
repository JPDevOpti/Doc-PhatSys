# Solicitud de Pruebas Adicionales

En ocasiones, el análisis inicial de la muestra revela la necesidad de realizar estudios complementarios (como tinciones especiales o inmunohistoquímica) para llegar a un diagnóstico preciso. PathSys permite solicitar estas pruebas directamente desde el módulo de resultados.

## ¿Cuándo solicitar pruebas adicionales?

Utilice esta funcionalidad cuando:
*   Necesite confirmar un diagnóstico diferencial.
*   Requiera marcadores específicos para tipificar un tumor.
*   La muestra inicial sea insuficiente o requiera un reprocesamiento.

## Pasos para realizar la solicitud

Esta opción se encuentra disponible dentro del módulo de **Transcripción de Resultados**, antes de firmar el caso.

1.  **Ubique la sección:** Desplácese hacia la parte inferior del formulario de resultados, encontrará el apartado "Pruebas Complementarias".
2.  **Seleccione las pruebas:**
    *   Utilice el buscador para encontrar la prueba requerida (por nombre o código).
    *   Indique la **cantidad** necesaria.
    *   Puede agregar múltiples pruebas en una misma solicitud.
3.  **Justifique la solicitud:**
    *   Es obligatorio ingresar un **motivo** en el campo de texto (ej. "Se requiere tinción de Ziehl-Neelsen para descartar tuberculosis").
4.  **Enviar:**
    *   Haga clic en el botón **"Enviar Solicitud"**.

## ¿Qué sucede después?

*   El caso cambiará su estado (o generará una alerta) para que el personal administrativo o técnico procese la nueva orden.
*   Usted podrá continuar con el análisis una vez los resultados de estas nuevas pruebas estén disponibles en el sistema.
*   La solicitud quedará registrada en el historial del caso para fines de trazabilidad y facturación.
